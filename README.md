# 评判标准
测评方式分为两种。
1. one shot 纯 API 调用，测试模型本身的代码能力和技术偏好。
2. 通过最流行的 Coding Agent 产品，实测模型在代码工程中的能力。

One Shot Case：
1. TodoList 网页开发
2. 2D 游戏 fly bird 开发
3. 3D 物理模拟弹珠小球
4. 3D 烟囱拆除

Coding Agent Case：
1. TodoList 纯前端页面开发
2. 2D 游戏 fly bird 开发
3. 3D 物理模拟弹珠小球
4. TodoList Vue + Java 全栈开发
5. 开源项目 BUG fix
