下面是一个简单的 **To-Do List 看板**（Kanban-style）界面的 HTML 示例，使用了基本的 HTML、CSS 和 JavaScript 来实现新增、拖拽和删除任务的功能。 ### ✅ 功能包括： - 添加任务到不同阶段（待办、进行中、已完成） - 通过拖拽将任务移动到不同阶段 - 删除任务 --- ## 📄 HTML 代码：`index.html` ```html TodoList 看板

TodoList 看板

待办

进行中

已完成

``` --- ### 💡 说明： - 三个看板块（待办 / 进行中 / 已完成），每个板块有一个用于添加任务的输入框。 - 每个任务可以拖拽到其他列中。 - 点击“删除”可从页面上移除任务。 - 使用 JavaScript 实现增删拖拽逻辑。 - 无额外依赖（Vue、React 等），原生实现。 --- 如果你希望用框架（如 Vue、React）或加入本地存储功能，也可以扩展。需要我帮你加这些功能吗？
